# mother-journey

## Project setup
```
npm install
```

```
brew install mongodb
mkdir -p /data/db
sudo chown -R `id -un` /data/db
```

## Run database
```
mongod
```

### Start backend server (in a new terminal window)
```
npm start
```
### Start frontend server (in a new terminal window)
```
npm run serve
```

### Run your tests (database needs to be running)
```
npm test
```
